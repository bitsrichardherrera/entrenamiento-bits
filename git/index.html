Que es GIT?

Git es un software de control de versiones pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran numero de archivos de codigo fuente. Su trabajo es llevar registro de los cambios en archivos de comptadora y coodinar el trabajo que varias personas realizan sobre archivos compartidos.

Comandos

Git clone

Clona un proyecto de git en la direccion que le asignes y el creara la carpeta, solo debes colocar la url de tu repositorio a continuacion un ejemplo.

git clone https://github.com/bitsrichardherrera/entrenamiento-bits.git

Git Help

Muestra una lista con los comandos más utilizados en GIT.

git help

Git add

Agrega al repositorio TODOS los archivos y carpetas que estén en nuestro proyecto, los cuales GIT no está siguiendo, tambien puedes agregar de 1 a la vez.

git add -A

git commit

Hace commit a los archivos que indiquemos, de esta manera quedan guardados nuestras modificaciones, se debe que aclarar que para poder guardar nuestros cambios se debe dejar un comentario entre " ", acontinuacion un ejemplo.

git commit -m "comentario"

Git checkout

Crea un nuevo branch y automaticamente GIT se cambia al branch creado, clonando el branch desde donde ejecutamos el comando.

git checkout -b "Nombre Del Branch"

Git branch

Nos muestra una lista de los branches que existen en nuestro repositorio.

git branch

Git checkout

Sirve para moverse entre branches, en este caso vamos al branch que indicamos en el comando.

git checkout "Nombre Del Branch"

Git status

Nos indica el estado del repositorio, por ejemplo cuales están modificados, cuales no están siendo seguidos por GIT, entre otras características.

git status

Git push

Luego de que hicimos un git commit, si estamos trabajando remotamente, este comando va a subir los archivos al repositorio remoto, específicamente al branch que indiquemos.

git push origin "Nombre del Branch"

Git pull

Hace una actualización en nuestro branch local, desde un branch remoto que indicamos en el comando

git pull origin "Nombre Del Branch"
